# Gather Context

Before executing a research task, identify what context is needed:

- **Background**: Key concepts, field context
- **Prerequisites**: Is the paper accessible? What depth?
- **Related work**: Cited papers, author's other work

## Output

```
### Retrieved
[what was gathered]

### Gaps
[what couldn't be retrieved]

### Ready to proceed
yes / no
```
