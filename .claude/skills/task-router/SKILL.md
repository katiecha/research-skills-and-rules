# Task Router

Produce a routing card before execution:

```
task_type:    paper-reading | literature-survey | conversational
skill:        read-paper | freeform
```

## Required inputs

| Skill | Required |
|---|---|
| `read-paper` | Paper URL, PDF, or pasted content |
| `read-paper` (survey) | Topic or research question |
