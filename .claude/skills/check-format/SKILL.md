# Format Check

Check that the output matches the read-paper contract for the given pass level.

**Pass 1**: Category, Context, Correctness, Contributions, Clarity, Recommendation, References

**Pass 2**: Main Argument, Key Figures (table), Evidence Assessment, Technical Gaps, Summary, Recommendation

**Pass 3**: Virtual Re-implementation, Assumption Audit (table), Innovations, Weaknesses, Future Work, Final Assessment, Summary

**Survey**: Key Papers, Key Researchers, Top Venues, Research Themes, Open Problems, Reading Order

## Output

```
### Violations
[list or "None"]

### Verdict
Pass / Fail — [N] violations
```
